# Basic Chat

- P2P basic chat app running in mobile environment

- Stack: holepunch, bare, bare-kit, react-native-bare-kit, corestore, hyperswarm, blind-pairing, hyperdb, hrpc, autobase

## Usage

```shell
# user1: create room
npm i
npm run build
npm run ios  
# or npm run android

# user2: clone into separate dir + join room
npm i
npm run build
npm run ios
# or npm run android
```
